Recommended resources

The Odin Project.

This is a full-stack learning resource for web development. Very highly rated by the web-development community.

FreeCodeCamp

Learning resource for not only full-stack web-development, but also machine learning, Databases, Data visualization, quality assurance, python, data analysis, college algebra, english, C# and many other things.

A Deep Dive Into Modern Web Development - Full Stack open

Learn React, Redux, Node.js, MongoDB, GraphQL and TypeScript in one go! This course will introduce you to modern JavaScript-based web development. The main focus is on building single page applications with ReactJS that use REST APIs built with Node.js.

Css

Javascript

Freecodecamp - JavaScript Algorithms and Data Structures (Beta)

Freecodecamp is a very good resource to learn front end web development, you also build projects using javascript, using modern / popular javascript concepts.

Javascript.Info

Javascript.info is an all in one resource from beginner to advanced, also teaching good patterns while learning Javascript!

React

Scrimba - React

Learning react has never been easier. Scrimba combines project-based learning and video based learning with built in editor and real-time video code editing.
